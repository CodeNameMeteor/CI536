# Sniffer(WIP name)

Sniffer is a Lightweight Packet Sniffer and Analyzer. Built in Java using Pcap4j. It supports IPv4, IPv6 Packets. And supports TCP, UDP, HTTP and HTTPS communication. It supports PCAP to allow for compatibility with other packet analyser tools such as wireshark.

## Technologies Used 
* **Language:** Java (JDK 25)
* **Framework:** JavaFX
* **Build Tool:** Maven
* **IDE:** IntelliJ IDEA
* **Testing:** JUnit 5
